Lesson 1: Introduction to Python

Python is a powerful, beginner-friendly programming language that emphasizes readability and simplicity.

Why Python? - Easy to learn - Versatile (Web, AI, Automation, Data Science) - Huge community support

1. Writing Your First Python Program

Let's start with a basic print statement:

print("Hello, world!")

This command outputs **Hello, world!** to the screen.

2. Python Syntax & Indentation

Unlike other languages, Python uses indentation instead of curly brackets for defining blocks:

def greet(name):
    print(f"Hello, {name}!")

greet("Treva")

Proper indentation is **mandatory** in Pythonâ€”it ensures code structure and readability.

3. Variables & Data Types

Python can store different types of data using variables:

name = "Treva"  # String
age = 25  # Integer
height = 5.9  # Float
is_student = False  # Boolean

Each type serves a different role in programming.

4. User Input

Python allows users to enter input, making programs interactive:

user_name = input("Enter your name: ")
print(f"Welcome, {user_name}!")

5. Practice Challenge

Try creating a program that asks for a user's favorite color and prints a message including their input.

Hint: Use the input() function like in the example above!
